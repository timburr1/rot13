<br>Simple Caesar Cipher

## Prerequisites
* [**Node.js**](https://nodejs.org/)

## Run the server:
Open the project folder in VSCode, then launch a new terminal. Cd into project root, then install dependencies with 
```
$ npm install
```
Start the web server with
```
$ node app.js
```
